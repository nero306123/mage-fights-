# Mage Arena

A complete browser-based mage arena RPG in a single HTML file. No server, no build step — just open `index.html`.

## Play

Open `index.html` in any modern browser.

- **Move:** WASD / Arrow keys
- **Cast:** number keys 1–5 (aim toward the mouse)
- **Spell menu (slow-mo):** hold TAB
- **Potions:** Q / E / R
- **Block (Earth Mage):** hold SPACE

## Features

- 8 mage classes, each with a unique sprite, passive, class mechanic and 8 spells
- Spell mastery with upgrade paths, elemental debuff combos and combo-casting
- Kill streaks, relics, weapons/armor/potion shops, bounties, prestige
- Normal, Boss and Endless Dungeon modes with arena events and mini-bosses
- Full CSS/SVG/Canvas visuals — particles, screen shake, slow-mo, dramatic banners
- Auto-saves to localStorage
